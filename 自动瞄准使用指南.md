# 🎯 AI-Aimbot 自动瞄准使用指南

## 🚀 如何开启自动瞄准

### 1. 启动自动瞄准程序
在项目目录下运行以下命令之一：

```bash
# 推荐：使用 ONNX 模式（GPU 加速）
python main_onnx.py

# 或者使用 PyTorch 模式
python main.py

# 或者使用 TensorRT 模式（最高性能）
python main_tensorrt.py
```

### 2. 选择游戏窗口
程序启动后会显示所有打开的窗口列表：
```
=== All Windows ===
[0]: Steam
[1]: Discord
[2]: Valorant
[3]: CS2
...
```
输入对应游戏窗口的数字（例如输入 `2` 选择 Valorant）

### 3. 自动瞄准开始工作
- 程序会自动激活选择的游戏窗口
- 开始实时检测和自动瞄准
- 在终端中会显示 FPS 和 CPS 信息

### 4. 停止自动瞄准
按 **Q** 键退出程序（可在 config.py 中修改退出键）

## ⚙️ 重要配置选项

### 基本设置 (config.py)
```python
# 自动瞄准移动幅度 (0.1-1.0)
aaMovementAmp = 0.4

# 检测置信度 (0.1-1.0)
confidence = 0.4

# 爆头模式（瞄准头部）
headshot_mode = True

# 退出键
aaQuitKey = "Q"

# GPU 选择 (1=CPU, 2=AMD, 3=NVIDIA)
onnxChoice = 3
```

### 性能优化设置
```python
# 截图区域大小
screenShotHeight = 320
screenShotWidth = 320

# 显示视觉效果（调试用）
visuals = False

# 显示 CPS 信息
cpsDisplay = True
```

## 🔧 常见问题解决

### Q: 程序无法检测到游戏窗口？
A: 确保游戏以窗口模式或无边框窗口模式运行

### Q: 自动瞄准不够精确？
A: 调整 `aaMovementAmp` 值（降低数值使移动更精细）

### Q: 检测到太多误报？
A: 提高 `confidence` 值（例如从 0.4 提高到 0.6）

### Q: 性能不够好？
A: 确保 `onnxChoice = 3`（使用 NVIDIA GPU）
